# Basic Gulp Template


### Gulp tasks
 - vendor prefixes
 - CSS minification
 - Image optimization

### Installation
You must have [Node.js](https://nodejs.org/) installed to use Gulp.

Install the dependencies and devDependencies and start the Gulp.

```sh
$ git clone repo_url
$ cd repo_name
$ npm i
```

### Start Gulp
```sh
$ gulp watch
$ gulp task_name
```
